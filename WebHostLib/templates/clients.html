Client Download

You can find the Desktop MultiworldGG Client for download below.

It can be used to connect to Multiworlds that do not provide their own custom client or to generate worlds locally.

The provided releases are built from the public source code available on Github.

Unfortunately, we cannot provide a feature complete OSX client at this point. Certain games won't work for now due to incompatible libraries. (e.g. Metroid Prime, Luigi's Mansion)

If you get an "App is damaged" message, please enter the following command in the terminal before starting again: xattr -c "/Applications/MultiworldGG.app"
